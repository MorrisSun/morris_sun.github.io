2019书单、观影、游戏记录

发表于 2019-04-20 | 分类于 记录生活 | 阅读次数：

2019年的第一次更新

  • #书
    • 暂无
  • #电视剧、电影
    • 古董局中局
      • 改编的有点垮，一开始还挺好看的，后来追不下去了。
    • 无双
      • 看完热血沸腾，情节紧凑，反转出人意料。
    • 来电狂响
      • 对人性的剖析，题材角度很有新意，个人感觉有点坑没有填上。
    • 神奇动物：格林德沃之罪
      • 小雀斑很帅，嘻嘻。
    • “大”人物
      • 包贝尔演技有点炸裂，情节也比较紧凑，总的来说不错。
    • 触及真心
      • 想要甜甜的恋爱。
    • 都挺好
      • 题材扎心，但是改编的时候可能为了大团圆的结局和刻意制造的矛盾点，有点烂尾。姚晨、郭京飞演技不错。
    • 绿皮书
      • 对于有色人种歧视有了深刻的反思，整个态度的变化是默默的，不突兀不做作，很多小细节处理的很好，配乐也perfect。
    • 飞驰人生
      • 韩寒一贯的风格，细节很有风格，彩蛋令人欣喜。
    • 疯狂的外星人
      • 宁浩的天马行空，两大喜剧扛把子的合体并没有大放光彩。给人感觉有点失望，但是没有网友说的那么惨。宁浩如果可以稍微收敛一点，两大影帝只留一个或许效果会更好。
    • 流浪地球
      • 很燃很炸，剧情、特效不错，但有点吴京化，情节有部分缺陷。
    • 今夜在浪漫剧场
      • 看完想恋爱，女主贼漂亮
    • 大黄蜂
      • 情节寡淡、特效感觉也一般，不过大黄蜂是蛮可爱。
    • 新喜剧之王
      • 无厘头看多了，可能就没啥感觉了，但是笑点还是有的。并且传达的是星爷一直坚持的，小人物也有大梦想。
    • 泡菜爱上小龙虾
      • 一般，打发时间。
    • 海王
      • 帅炸
    • 调音师
      • 反转反转再反转，人性本恶。
    • 反贪风暴3
      • 相貌平平古天乐。
    • 大地
      • 人文课作业，一开始会觉得很无聊，但最后看下来比较惊喜。影片改编自赛珍珠的小说《大地》，通过一对农民夫妻的命运，展示了中国农民身上的优点。导演通过一些小细节来刻画中国人的特点，很用心。
